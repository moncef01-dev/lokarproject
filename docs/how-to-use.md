# How to Use the Lokar System

This document explains the core workflow and features of the Lokar car rental management system.

## 1. System Roles & Access Control

The system follows a hierarchical structure for managing car rentals and agencies.

### Superadmin Workflow

The Superadmin has the highest level of authority and manages the platform's ecosystem.

- **Adding Agencies**: From the Admin Panel, the Superadmin can promote existing registered users to Agency status.
- **Process**:
  1. The user must first create a regular account.
  2. The Superadmin enters the user's **email address** and assigns an **Agency Name**.
  3. Once converted, the user gains access to the Agency features.

### Agency Dashboard

Once a user is designated as an agency, they get access to a dedicated dashboard.

- **Management**: Agencies can manage their fleet, handle bookings, and view analytics.
- **Adding Cars**: Agencies can list new vehicles available for rent through their dashboard.

## 2. Managing Car Listings

The system is designed to be lightweight by using external image references or direct uploads.

### Car Images

To optimize storage and simplify the process of adding cars:

- **External URLs**: You can use high-quality image URLs from the web.
- **Direct Upload**: Support for uploading vehicle images directly to the system.
- **Benefit**: Keeps the system fast while providing flexibility in how you showcase your fleet.

## 3. Booking Management & Contracts

Agencies can manage customer prebookings and generate legal contracts seamlessly.

### Booking Workflow
1. **View Requests**: Navigate to the **Bookings** tab to see all customer prebooking requests.
2. **Search & Filter**: Use the search bar to find specific customers by name, email, or phone.
3. **Generate Contract**: Click the **Generate PDF** button on any prebooking.
4. **Customization**: A modal will appear allowing you to override contract details (e.g., specific terms or pricing).
5. **Download**: Once generated, the PDF contract is automatically downloaded for signing.

## 4. Analytics & Reporting

The **Analytics** tab provides real-time insights into your agency's performance.

- **Revenue Chart**: Track your earnings over time with interactive line charts.
- **Vehicle Performance**: See which cars in your fleet are performing best (highest rental counts).
- **Key Metrics**: Quick-glance cards showing Total Profit, Fleet Size, and Total Rentals.

## 5. General Workflow Summary

1. **User Registration**: A user signs up on the platform.
2. **Agency Creation**: Superadmin upgrades the user to an agency via the Admin Panel.
3. **Fleet Management**: The new agency logs in, adds cars, and sets availability.
4. **Booking Handling**: Agency manages incoming requests and generates rental contracts.
5. **Performance Review**: Agency uses the Analytics dashboard to optimize their business.
